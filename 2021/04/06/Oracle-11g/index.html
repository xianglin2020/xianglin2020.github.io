安装步骤

Docker 安装步骤

  1. 删除旧版 Docker

  2. 安装依赖

  3. 导入 Docker GPG 公钥

  4. 添加软件仓库

  5. 安装 Docker

  6. 替换国内源，编辑/etc/docker/daemon.json

  7. 将当前用户加入 docker 组

Oracle-11g 安装步骤

  1. 下载 linux.x64_11gR2_database_1of2.zip、linux.x64_11gR2_database_2of2.zip两个文件，将其解压。

  2. 拉取 Oracle-11g 的 docker 镜像

  3. 安装启动 Oracle Database

    安装过程较长，查看输出，有 100% complete 时，表示安装成功。

解除账户限制

  1. 连接到容器

  2. 切换到 oracle 用户，然后链接到 sql 控制台

  3. 解锁账户

  4. 使用可视化工具登录
