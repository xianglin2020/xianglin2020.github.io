使用 APT 安装 MySQL

  1. 使用命令安装 MySQL Server

    sudo apt install mysql-server

  2. 查看 MySQL 运行情况

    sudo service mysql status

    或者

    sudo /etc/init.d/mysql status

    启动、停止、重启MySQL 的命令类似

  3. 在 Ubuntu 中安装 MySQL 并没有提示设置 root 密码，但可以在/etc/mysql路径下找到相关配置文件

    会得到如下信息：

    可以使用debain.cnf中的 user和password登录 MySQL

  4. 新建一个用于远程连接的用户，并分配权限

    新建用户

    create user 'xianglin'@'119.130.215.117' identified by '950915';

    分配权限

    grant all privileges on *.* to 'xianglin'@'119.130.215.117';

    flush privileges;

  5. 找到/etc/mysql/mysql.conf.d/mysqld.cnf并注释bind-address=127.0.01，后重启 MySQL 服务
