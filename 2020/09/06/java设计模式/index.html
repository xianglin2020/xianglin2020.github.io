设计模式

  • 设计模式是前人总结的设计经验
  • 设计模式的目标是代码更容易理解和维护
  • 设计模式可以让代码更可靠

工厂模式

  • 工厂模式用于隐藏创建对象的细节
  • 工厂模式的核心：工厂类（factory）
  • 工厂模式可细分为：简单工厂、工厂方法、抽象工厂
